2. Document Overview

The purpose of this user manual is to document the functionalities and capabilities of the OSNAP application. This manual is accessible to OSNAP users once they have authenticated and successfully logged in to the application.

The following conventions apply throughout this manual:

  1. Bold typeface is used to denote the following elements of the RESHAPE TDA user interface:
    1. Buttons
    2. Checkboxes
    3. Field names
    4. Tabs
    5. Sub-tabs
  2. Italic typeface is used for hyperlinks, icons and any other special references on the screen within the OSNAP user interface.
  3. Quotation marks (“ ”) indicate the title of a screen name or window within OSNAP.
